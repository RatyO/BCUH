BCUH

BCUH is a collection of simple univariate bi-variate methods for combining information obtained from reference data with (typically) climate simulations. These methods have become de facto standard in climate research due to the need to take biases in climate model simulations with respect to the observed climate into account.

Introduction

This tutorial illustrates the use of R package BCUH in which several bias correction and delta change methods have been implemented following [raisanen2013] and [raty2014]. In addition, a copula-based joint bias correction method designed for temperature and precipitation is available. The implementation of the copula-based algorithm is partially based on [li2014] with modifications from [gennaretti2015].
